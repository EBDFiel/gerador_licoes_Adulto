Gerador de lições EBD Fiel

Este é o seu gerador bonito da EBD Fiel já preparado para trabalhar com backend Node.js/Express. Os textos originais da revista permanecem preservados no formulário, enquanto a IA gera análise geral, apoio pedagógico e aplicação prática por meio de uma rota segura.

Mesmo visual da EBD Fiel

O gerador mantém a mesma identidade visual aprovada, com blocos elegantes, contraste forte e a mesma sensação visual do seu site.

Integração segura com backend

O frontend chama o backend em /api/gerar-complementos, sem expor chave de API no navegador.

Formulário da lição

Os textos originais da revista ficam preservados. A IA gera somente análise geral, apoio pedagógico e aplicação prática.

Integração com IA: cole o texto bruto da revista, monte a estrutura e use os botões de IA para preencher automaticamente apenas os complementos. Se o campo de URL da API estiver vazio, o gerador usa a mesma origem da página atual.

Configuração da integração
Deixe vazio para usar a mesma origem da aplicação.
A montagem automática funciona melhor quando o texto contém marcações como Introdução, Conclusão, 1., 1.1., 2., 2.1. etc.
